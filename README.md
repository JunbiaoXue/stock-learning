  文件：04-基本面分析/

第5阶段：风控与心态（1周）
  目标：建立纪律，不再亏大钱
  内容：仓位管理、止损纪律、心态调整

第6阶段：实盘训练（持续）
  目标：在天邦食品上实战做T，降成本解套
  内容：每日复盘、做T记录、策略调整
  文件：06-实盘训练/


学习方式

每个阶段包含：
1. 知识点讲解（我生成）
2. 案例分析（结合天邦食品实盘）
3. 练习任务（你完成）
4. 阶段测试（我出题你答）

完成一个阶段再进入下一个，不赶进度。
