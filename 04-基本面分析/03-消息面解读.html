📰 第14课：消息面解读

A股受消息影响很大。利好不涨要小心，利空不跌是机会。这课教你分辨消息真假，判断消息对股价的影响。

一、消息类型与影响

消息类型影响强度持续时间做T应对
国家政策🔴🔴🔴 极强1-2周顺势做T，不逆势
行业政策🔴🔴 很强3-7天板块联动，选龙头
公司公告🔴 中等1-3天看公告性质决定
龙虎榜🟡 偏短1-2天游资票波动大，小心
北向资金🟢 中等持续性连续流入=看好
小道消息❌ 不确定不确定不参考！

二、天邦食品核心消息面

📐 当前最关键的3个消息

1. 预重整延期至2026年11月

→ 不确定性增加，但还没失败，悬着

→ 预重整成功=大利好，失败=大利空

2. 猪价持续低迷

→ 短期利空，但猪周期底部区域

→ 关注能繁母猪存栏数据（减少=利好）

3. 退市风险

→ ST风险存在但预重整中暂安全

→ 如果预重整失败可能被ST，股价大跌

（数据截至2026年5月18日）

四、龙虎榜与北向资金

🐉 龙虎榜——看谁在买卖

龙虎榜披露大额买卖的营业部，帮你判断主力动向：

机构买入 → 中长期看好，可跟随

知名游资买入 → 短线炒作，波动大，做T机会多

机构卖出 → 中长期看空，减仓

拉萨天团（东方财富拉萨营业部） → 散户大本营，不跟

🌐 北向资金——聪明钱的方向

北向资金 = 从香港流入A股的资金，被称为"聪明钱"：

连续3天以上净流入 → 看好，可做多

连续3天以上净流出 → 看空，减少做T

查看方式：东方财富网站 → 北向资金

📝 本课要点

  • 消息影响：国家政策>行业政策>公司公告>龙虎榜>北向资金
  • 天邦核心消息：预重整进展、猪价走势、退市风险
  • 利好不涨=利好出尽（卖），利空不跌=利空出尽（买）
  • 利好大涨=做反T好时机，利空大跌=等企稳再抄底
  • 龙虎榜看主力动向，北向资金看聪明钱方向
  • 小道消息不参考，只看正式公告和数据
